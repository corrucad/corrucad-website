BoxFinder compares standard and bulk prices across 12+ top industrial vendors in real-time, helping e-commerce brands and shippers stop overpaying for boxes.✨ Key Features🔍 BoxFinder (Metasearch Engine)Real-Time Aggregation: Compares prices from Uline, PaperMart, Grainger, The Boxery, Staples, Zoro, Packaging Hero, and more.Smart Tiering: Toggles instantly between Standard (<100 units) and Bulk/Bale (800+ units) pricing matrices.Affiliate Integration: Seamlessly monetized via partnerships with Shippo (shipping software) and Arka (custom packaging).📐 CorruCAD (Estimator)Free FEFCO box style dimension calculations and structural estimations.🚀 Programmatic SEO ArchitectureInstead of relying on a single Single Page Application (SPA) routing system, this repository utilizes a custom programmatic SEO build pipeline to dominate organic search:Node.js Build Script (build.js): Ingests a master array of box dimensions and vendor prices (data.js) and injects them into a master template.html.Static Generation: Automatically stamps out 50+ hyper-optimized, standalone HTML landing pages (e.g., 12x12x12-corrugated-boxes.html).Rich Snippet Injection: Dynamically generates SoftwareApplication and AggregateOffer JSON-LD Schema markup for every single page to secure "Position Zero" and price snippets in Google Search and AI Overviews.Automated Sitemaps: Generates an up-to-date sitemap.xml on every build.🛠 Tech StackFrontend: React 18 (via CDN), Tailwind CSS, Babel (Standalone)Build Step: Node.js (Vanilla FS module for templating)CI/CD & Hosting: Vercel💻 Local Development & Deployment The magic of this repository happens in the build script.To update prices or add new box sizes:Open the source data array.Update the vendor pricing or add a new dimension object.Commit and push to GitHub.The Vercel Pipeline:Upon pushing to the main branch, Vercel automatically runs node build.js.
